# Geometry-Aware Transform Pruning for Omnidirectional Image Compression

This repository contains the source code associated with the paper:

**“Geometry-Aware Transform Pruning for Omnidirectional Image Compression”**  
submitted to *IEEE Access*.

## Authors

- Thiago L. T. da Silveira  (UFSM)
- Enzo G. Segala  (UFRGS)
- Fábio M. Bayer  (UFSM)
- Renato J. Cintra  (UFPE)

## Overview

This project investigates a geometry-aware transform pruning framework for efficient compression of omnidirectional (360°) images. The proposed method leverages structural redundancies in transform domains to reduce computational and representational complexity, while preserving visual fidelity and perceptual quality.

## Code Availability

The source code will be made publicly available upon acceptance and publication of the paper in *IEEE Access*.
