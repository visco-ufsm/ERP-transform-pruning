# Sample Image Disclaimer

The image `PoleVault_le_3840x1920_30fps_8bit_420_erp_0.bmp` is provided only as a sample input for testing the implementation.

This image corresponds to the first frame of an uncompressed omnidirectional video sequence defined in the Common Test Conditions (CTC) for 360° video coding. Due to licensing and distribution restrictions, the complete video sequences cannot be redistributed with this repository.

The provided frame is included to facilitate reproducibility and to demonstrate the execution of the proposed geometry-aware transform pruning method. For a complete evaluation, users should obtain the original uncompressed 360° video sequences from the corresponding CTC dataset and extract the required frames for testing.
